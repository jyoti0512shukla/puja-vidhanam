# Story Images (Sadhana App)

Place cover and section images here for downloadable Katha stories.

- **Cover images**: `{coverImage}.png` (e.g. `my-new-story.png`)
- **Section images**: `{id}-01.png`, `{id}-02.png` (e.g. `my-new-story-01.png`)

Bundled Panchatantra stories use images shipped in the app. Add images here only for stories with `bundled: false` in `kathas/katha-catalog.json`.

See `docs/ADD_NEW_KATHA.md` in the Sadhana repo for the full guide.
